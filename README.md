Welcome to JobHunt, Your one-stop employment website constructed with state-of-the-art MERN stack technology. Seamlessly connect recruiters and applicants through secure login, enhance interviews with video chat, craft impressive resumes, and manage job postings effortlessly. ## Features

- User Authentication (Recruiter/Applicant login)
- Video Chat
- Resume Building
- Jobs Management (Create, Update, Delete)
- Dashboard (Separate views for recruiters and applicants)
- User Profiles
- Analytics and Reporting
- Responsive Design
- Security (HTTPS, data protection)

## Technology used

- ReactJS
- NodeJS
- Express
- MongoDB
- Redux

## To run locally on your machine

### Clone the repo to your machine

```
git clone https://github.com/shubhamrupapara/job-hunt.git
```


### Install Dependencies (frontend & backend)

```
cd BACKEND
npm install
cd FRONTEND
npm install
```

# Running the application

```
# Run frontend (:3000) & backend (:5000)
npm start

# Run backend only
npm start
```
